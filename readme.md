# Panduan Penggunaan

1. Clone repositori ini dengan `
 git clone https://github.com/mcqeems/cs-ticket-mid.git`
2. Kemudian jalankan `pip install -r requirements.txt`
3. Jalankan program dengan `python cs_ticket.py`

> Apabila menggunakan virtual environment maka beda gunakan perintah yang berbeda.
